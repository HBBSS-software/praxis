# 可爱奶龙社会实践系统

## 技术栈

- 运行时：Node.js
- 前端：React + Vite + shadcn/ui
- 后端：Node.js + Hono
- 数据库：SQLite + Drizzle ORM

## 如何使用

安装依赖：

```bash
pnpm install --frozen-lockfile
```

启动：

```bash
pnpm start
```

更多命令见 `package.json`。

## 配置

见 `config.toml`，首次启动会自动生成。
